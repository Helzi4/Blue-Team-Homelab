# Scenario

A user reported a suspicious Gmail message themed as a shared invoice document.

The email contained a link presented as a document access button. After opening the message, the user clicked the link and was redirected to an internal decoy web page hosted in the lab.

The goal of the investigation was to determine:
- whether the user interaction actually occurred,
- whether any credential submission or follow-on activity took place,
- and whether the case should be treated as phishing interaction only or as confirmed compromise.
