# Remediation

## Lab Decision
No containment action was required because this was a controlled internal simulation.

## Production-Oriented Response
In a real environment, the following actions would be appropriate:
- validate the ownership and purpose of the source host
- determine whether the source is an approved scanner or admin system
- review whether SMB / RDP exposure is necessary
- investigate any follow-on authentication or process execution
- isolate the source host if the activity is unauthorized and continues
